# Chrome Extension for NotebookLM

A Chrome extension that enhances NotebookLM with advanced organization features including smart folders, tags, and metadata management.

## Project Status

This project is currently in development. See [docs/tasks.json](docs/tasks.json) for the current task breakdown and progress.

## Development Phases

- **Phase 0**: Foundation & TDD Setup
- **Phase 1**: The Viewer and Discovery
- **Phase 2**: The Parasitic Sidebar
- **Phase 3**: Metadata and Storage
- **Phase 4**: The Logic Engine (Smart Folders)
- **Phase 5**: Sync and Polish
